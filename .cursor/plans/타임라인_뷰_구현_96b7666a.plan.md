- 컨테이너: `display: flex; overflow-x: auto; gap: $spacing-md; padding-bottom: $spacing-md`

## 2. TaskList.vue 수정

### 헤더 버튼 추가

```html
<!-- 완료 리스트 복사 버튼 옆에 추가 -->
<button
  type="button"
  class="task-list__timeline-btn"
  :class="{ 'task-list__timeline-btn--active': showTimeline }"
  @click="toggleTimeline"
>
  타임라인
</button>
```

### 스크립트

```typescript
const props = defineProps<{ selectedTaskId?: string; showTimeline: boolean }>();
const emit = defineEmits<{
  (e: "select", taskId: string): void;
  (e: "delete", taskId: string): void;
  (e: "toggle-timeline"): void; // 추가
}>();

function toggleTimeline() {
  emit("toggle-timeline");
}
```

### 콘텐츠 전환

```html
<!-- 기존 main-outer / export-outer 영역을 v-if로 감싸기 -->
<template v-if="!showTimeline">
  <!-- 기존 task-list__main-outer, export-outer 전부 -->
</template>
<TaskTimeline
  v-else
  :selected-task-id="selectedTaskId"
  @select="emit('select', $event)"
/>
```

## 3. HomeView.vue 수정

### 스크립트

```typescript
const showTimeline = ref(false);

function handleToggleTimeline() {
  showTimeline.value = !showTimeline.value;
}
```

### 템플릿

```html
<!-- tasks col에 prop + emit 추가 -->
<TaskList
  :selected-task-id="selectedTaskId"
  :show-timeline="showTimeline"
  @select="handleSelect"
  @delete="handleDelete"
  @toggle-timeline="handleToggleTimeline"
/>

<!-- matrix: Transition + v-if -->
<Transition name="matrix-exit">
  <section v-if="!showTimeline" class="app__col app__col--matrix">
    <PriorityMatrix ... />
  </section>
</Transition>
```

### SCSS


## 동작 흐름

```mermaid
sequenceDiagram
    actor User
    participant TL as TaskList_헤더
    participant HV as HomeView
    participant Matrix as PriorityMatrix
    participant TT as TaskTimeline

    User->>TL: "타임라인" 버튼 클릭
    TL->>HV: emit('toggle-timeline')
    HV->>HV: showTimeline = true
    HV->>Matrix: v-if=false → Transition leave (우측 slide out)
    HV->>TL: :show-timeline=true
    TL->>TT: v-if=true 렌더링
    User->>TT: 태스크 카드 클릭
    TT->>TL: emit('select', taskId)
    TL->>HV: emit('select', taskId)
```

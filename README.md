# Mandarin Priority

감정이 아닌 수치로 업무 우선순위를 시각화하는 도구

## 개요

Mandarin Priority는 중요도와 시급성을 별점으로 입력받아 2x2 매트릭스로 시각화하여 업무 우선순위를 명확하게 파악할 수 있게 도와주는 웹 애플리케이션입니다.

## 주요 기능

- ✅ 업무 항목 생성/수정/삭제
- ✅ 인라인 편집 (업무 항목 확장하여 바로 수정)
- ✅ 2가지 별점 입력 시스템 (중요도, 시급성)
- ✅ 자동 점수 계산 및 사분면 분류
- ✅ 2x2 매트릭스 시각화
- ✅ 실시간 업데이트

## 기술 스택

- Vue 3 (Composition API)
- TypeScript
- Pinia (상태 관리)
- SCSS (스타일링)
- Vite (빌드 도구)

## 사용 방법

### 업무 추가
1. 좌측 상단 "추가" 버튼을 클릭합니다.
2. 헤더 바로 아래 나타나는 폼에 제목과 설명을 입력합니다.
3. 2가지 항목에 대해 별점(1~5점)을 입력합니다:
   - 중요도 (1~5점)
   - 시급성 (1~5점)
4. "추가" 버튼을 클릭하여 저장합니다.
5. 우측 매트릭스에서 업무가 자동으로 배치된 것을 확인합니다.

### 업무 수정
1. 업무 목록에서 수정하려는 항목의 "수정" 버튼을 클릭합니다.
2. 해당 업무 항목이 확장되어 편집 폼이 나타납니다.
3. 내용을 수정한 후 "저장" 버튼을 클릭합니다.

### 업무 선택 및 삭제
- 업무를 클릭하여 선택하면 매트릭스에서 강조 표시됩니다.
- "삭제" 버튼을 클릭하여 업무를 제거할 수 있습니다.

## 사분면 설명

- **A (중요↑ 시급↑)**: 즉시 처리해야 할 업무
- **B (중요↑ 시급↓)**: 계획적으로 처리할 업무
- **C (중요↓ 시급↑)**: 위임하거나 최소화할 업무
- **D (중요↓ 시급↓)**: 제거 대상 업무

## 점수 계산 방식

- **중요도 점수** = 중요도 - 3 (-2 ~ +2, 3점이 기준점 0)
- **시급성 점수** = 시급성 - 3 (-2 ~ +2, 3점이 기준점 0)
- 1점 = -2, 2점 = -1, 3점 = 0, 4점 = +1, 5점 = +2
